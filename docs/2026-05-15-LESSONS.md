## Pattern recognition across L37–L44

Today's eight lessons cluster into three meta-themes worth naming:

### Theme A: Closed-bar vs live-tick is a TWO-MODE problem, not a one-size-fits-all rule

L37 (closed-bar lag) and L41 (SHOTGUN_SCALPER birth) are the same lesson at two scopes. L34 fixed a CORRECTNESS bug (in-progress bars look closed). It did not address a STRUCTURAL latency tradeoff. The doctrine emerging from today: **5m closed-bar discipline is correct for ride-the-ribbon trades; live-tick triggers are correct for scalp-rejection trades; the two coexist in the playbook, not compete.** Per OP 23 (SNIPER_LEVEL_BREAK already bypasses ribbon-spread gate for ★★+ breaks), this is the natural extension. The mental model: the engine has a HIGH-CONVICTION-NAMED-LEVEL branch and a STANDARD-CONFLUENCE branch. They have different gates, different sizing, different exits. Conflating them is the failure family.

### Theme B: Documentation, memory, lessons, params, and prompts are NOT interchangeable surfaces

L40 surfaced this explicitly. L42 and L43 reinforced it. The OP 4 "no code drift" rule is bidirectional but asymmetric: a prompt change requires a lesson; a lesson does NOT require a prompt change. When I report fix status to J, the verb chosen MUST signal which surface was edited. Vocabulary lock from today: "documented" (docs only), "lesson logged" (LESSONS.md only), "queued" (in `automation/overnight/queue.md`), "wired" (params.json or lib/ edited), "encoded" (prompt edited AND verified active). Each verb is checkable — `git diff` or `Get-Content` will confirm. Misusing the verb is the silent-failure mode this session almost shipped.

### Theme C: The EOD pipeline is the source-of-truth surface — bugs there are doctrine bugs

L42 (equity=0 + copy-pasted counterfactuals) and L43 (no missed-setup enumeration) are not isolated bugs. They are **structural gaps in the surface J uses to grade the engine**. If the EOD pipeline is wrong, J's grading is wrong, J's edge measurement is wrong, the grinder's J-anchor weighting is wrong (per OP 16). The EOD pipeline is therefore as high-priority as the heartbeat itself. Today's queued fixes (T## tonight) are not "polish" — they're "fix the measurement instrument."
