## How to extend this library

When a new futures question comes up:
1. Check CME Group first for anything structural (specs, hours, expiration, position limits).
2. Check the broker (Tastytrade) for anything account-specific (margins, commissions, order routing).
3. Use Green Trader Tax / a CPA for tax.
4. Add the finding to the relevant file here + append the source URL to this list with a retrieval date.
