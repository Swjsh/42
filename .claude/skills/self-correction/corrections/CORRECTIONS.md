# Corrections memory

> This is your durable correction memory. Each block is one standing rule the user gave you.
> Read every entry at the start of every session and follow it. When the user corrects you, append a
> new block here (see the self-correction SKILL.md). The user owns this file — they can edit or delete
> any rule, and their edit wins over your memory.
>
> Format per entry:
> ```
> ## C# — short title
> - **Rule:** the behavior to follow.
> - **Why:** the reason (so you can judge edge cases).
> - **Applies when:** the situation that triggers this rule.
> - **Trigger phrases:** the words the user used.
> - **Added:** YYYY-MM-DD
> ```
>
> Never store secrets, credentials, keys, or personal data here — record the *rule* generically.
